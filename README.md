## Comando para tener spotify sin anuncios
> [!IMPORTANT]
> Descargar la versión de spotify de la pagina de spotify y no en la tienda de microsoft. 
> Para ello se debe ir al siguiente link: 
> ``` https://www.spotify.com/es/download/windows/ ```

><img width="1365" height="597" alt="image" src="https://github.com/user-attachments/assets/b7af8c4d-9c51-4427-bc58-12e565b50732" />

> [!NOTE]
> Una vez descagargado entra y registrate con tu cuenta y una vez registrado cierra espotify
>

> [!IMPORTANT]
> Abrir la terminal de  ```Powershell``` y ejecutar el siguiente cómando
> ```bash
> iwr -useb https://raw.githubusercontent.com/spicetify/cli/main/install.ps1 | iex
> ```
> su documentación se encuentra en ```https://spicetify.app/docs/getting-started/```

> [!NOTE]
> Una vez hecho los pasos anteriores y ya descargado entra a lo que es spotify y dale volve a cargar y te aparecera una tienda en la parte izquierda, una vez ahí buscas addblock spotify premium y listo recargas y no saldran anuncios nuevamente
